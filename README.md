# Credit-Card-Fraud-Detection-System-for-Sri-Lanka-Using-Machine-Learning
Credit card fraud is one of the fastest‑growing financial crimes worldwide, and Sri Lanka is experiencing a sharp rise in fraudulent activity with the rapid adoption of e‑commerce and digital banking. Traditional fraud prevention mechanisms in the country remain largely rule‑based, manual, and reactive, which makes them ineffective against modern, sophisticated fraud techniques. This project introduces a hybrid fraud detection system that combines SMS‑based user authentication with machine learning (ML) models tailored to the Sri Lankan financial sector.
